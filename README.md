## 📦 Installation

### Step 1: Add JitPack repository

gradle
dependencyResolutionManagement {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}




### Step 2: Add dependency

gradle
implementation 'com.github.afzal-hussain-coder:LibrarayDemo:v1.0.0'
